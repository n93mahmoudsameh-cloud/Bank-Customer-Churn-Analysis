# Bank Customer Churn Analysis Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard designed to analyze customer churn in a banking environment. The dashboard helps identify customer segments with higher churn rates and provides insights to support customer retention strategies.

---

## 🎯 Business Objective
The goal of this project is to analyze customer behavior and identify the key factors influencing customer churn.

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📊 Dashboard Features
- Customer Churn Rate
- Churn Rate by Age Group
- Churn Rate by Gender
- Churn Rate by Geography
- Churn Rate by Active Membership
- Customer Status Overview
- Interactive Filters

---

## 💡 Key Insights
- The overall churn rate is **20.37%**.
- Customers aged **46–60** have the highest churn rate (**51.12%**).
- Female customers show a higher churn rate than male customers.
- Germany records the highest churn rate among all regions.
- Inactive customers are significantly more likely to churn than active customers.
